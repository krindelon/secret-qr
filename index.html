How it works (Option 3): We create exactly one resident credential on your physical security key. For each "Secret name", we compute salt = SHA-256("v1:" + name) and call navigator.credentials.get() with prf.eval.first = salt. The authenticator returns a deterministic 32-byte PRF output bound to that credential + salt. We use that output directly as the AES-GCM-256 key. No keys are stored in the browser—only the encrypted payload.
